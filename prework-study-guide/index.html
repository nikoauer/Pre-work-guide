  • Git is a versional control software that allows users to monitor the varying stages of code over time

    • creates a new branch and switch to it
  • Git add -A: stages the files to be added
  • Git commit -m "changes made to file": is used to commit changes to the file along with brief description of what changes have been made
  • Git pull origin main: pulls the up to date source code
  • Git push origin name of new branch: pushes the source up to github
